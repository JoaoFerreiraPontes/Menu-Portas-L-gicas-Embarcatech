# MENU DE PORTAS SIMULAÇÃO DE PORTAS LÓGICAS EM C++ SDK
**Este projeto tem como finalidade simular a ação e comportamento de 7 portas lógicas, sendo elas:**
 - AND
 - OR
 - NOT
 - NAND
 - NOR
 - XOR
 - git XNOR
